#  Tableau Superstore Analytics Dashboard

This project presents an interactive Tableau dashboard built using the Superstore dataset to analyze key business metrics such as Sales, Profit Margin, Return Rate, Regional Performance, Category Trends, and Product Insights. The dashboard helps identify sales patterns, profitable categories, regional opportunities, and customer segment contributions.

---

##  Key Features


- Visualizes key KPIs: **Sales, Profit Margin, and Return Rate**.  
- Shows **category-wise and sub-category sales performance**.  
- Provides **state-level and regional sales insights** using an interactive map.  
- Analyzes **customer segment contribution** through a donut chart.  
- Highlights **top products by sales**.  
- Displays **sales trend over time** with a clear line chart.



---

##  Tools & Techniques Used
- **Tableau Desktop**
- Data Cleaning & Filtering
- Dashboard Design & Layouting
- Trend Analysis & Storytelling
- Interactive Filters (Metric, Region, Year, etc.)

---

##  Dataset
This project uses the **Superstore dataset**, a widely used sample dataset for data analytics and visualization practice.

---

##  Purpose of the Dashboard
- To analyze Superstore's sales and profit performance.
- Identify trends across categories, segments, products, and regions.
- Provide actionable insights for business growth and strategy.
- Demonstrate Tableau dashboarding and data storytelling skills.
